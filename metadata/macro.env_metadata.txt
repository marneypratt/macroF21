---------------------
DATA & FILE OVERVIEW
---------------------

This dataset is a join of the macro and env datasets. If there was more than one water quality measurement taken with in a microhabitat on the same day, then those values were averaged. Otherwise, all the columns are described in the macro_metadata.txt and env_metadata.tct files.
